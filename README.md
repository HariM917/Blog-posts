# Blog Platform with Archive Data

A modern full-stack blogging platform populated with an archive of over 139,000 blog posts.

## Features
- **Modern UI**: Clean, editorial-style design with elegant typography.
- **Massive Dataset**: Over 139,000 posts and 1,900 users imported from a historical blog archive.
- **Optimized Performance**: Backend fetches are limited to 50 posts per request to ensure fast loading times.
- **User Authentication**: Secure JWT-based registration and login.
- **Comment System**: Interactive community features for every post.

## Tech Stack
- **Frontend**: React, Lucide Icons, Axios.
- **Backend**: Node.js, Express, MongoDB (Mongoose).

## Getting Started

### Prerequisites
- Node.js installed.
- MongoDB Atlas account.
